1. KineticJS is an HTML5 Canvas JavaScript framework

  2. It enables high performance animations, transitions, node nesting, layering, filtering and caching

  3. It enables event handling for desktop and mobile applications,

  4. You can draw things onto the stage, add event listeners to them, move them, scale them, and rotate them independently from other shapes to support high performance animations

How it Works?

  1. Kinetic stage is made up of user defined layers

  2. Each layer can contain shapes, groups of shapes, or groups of other groups. The stage, layers, groups, and shapes are virtual nodes, similar to DOM nodes in an HTML page

Features

  1. Object Oriented API

  2. Node nesting and event bubbling

  3. High performance event detection

  4. Layering support

  5. Nodes can be converted into data URLs, image data, or image objects

  6. Nodes can be converted into data URLs, image data, or image objects

  7. Animation and Transition support

  8. Drag and drop with configurable constraints and bounds

  9. Ready to use shapes including rectangles, circles, images, text, lines, polygons and custom shapes

  10. Selector support e.g. stage.get('#foo') and layer.get('.bar')

  11. Desktop and mobile events

  12. Event driven architecture which enables developers to subscribe to attr change events, layer draw events, and more

  13. And Much more...
