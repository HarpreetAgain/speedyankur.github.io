What is Phonegap?

  1. PhoneGap is an web-based mobile development framework

  2. Avoiding coading for each mobile platform in native development language like JAVA, Obj C, C/C++

  3. HTML5, CSS3, and JavaScript for cross-platform development

But why Cross-platform mobile using Phonegap?

  1. One code base, multiple platforms

  2. Reuse existing web developer skills

  3. Shorter learning curve

  4. Faster development

  5. Mobile site and app with same code

  6. Reduced QA efforts

  7. Active and growing community

Supported platforms

Available API

  1. Accelerometer : Captures device motion in the x, y, and z direction.

  2. Camera : The camera object provides access to the device's default camera application.

  3. Capture : Record audio and videos

  4. Compass : Obtains the direction that the device is pointing.

  5. Geolocation : The geolocation object provides access to location data based on the device's GPS sensor or inferred from network signals.

  6. Notification : Visual, audible, and tactile device notifications.

  7. Contacts : The contacts object provides access to the device contacts database.

  8. Device : The device object describes the device's hardware and software.

  9. Events : deviceready, pause, resume.
